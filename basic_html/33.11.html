Previously this formula was expressed as an equality. Since this formula expresses an asymptotic expansion, it has been corrected by using instead an ∼ relation.

Errata (effective with 1.0.19):
Originally the factor in the denominator on the right-hand side was written incorrectly as (∓2⁢i⁢ρ)k. This has been corrected to (±2⁢i⁢ρ)k.
