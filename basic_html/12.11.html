§12.11(i) Distribution of Real Zeros

ⓘ
Keywords:
distribution, parabolic cylinder functions, zeros
Notes:
See Whittaker and Watson (1927, p. 354), Dean (1966).
Permalink:
http://dlmf.nist.gov/12.11.i
See also:
Annotations for §12.11 and Ch.12

If a≥−12, then U⁡(a,x) has no real zeros. If −32<a<−12, then U⁡(a,x) has no positive real zeros. If −2⁢n−32<a<−2⁢n+12, n=1,2,…, then U⁡(a,x) has n positive real zeros. Lastly, when a=−n−12, n=1,2,… (Hermite polynomial case) U⁡(a,x) has n zeros and they lie in the interval [−2⁢−a,2⁢−a]. For further information on these cases see Dean (1966).

If a>−12, then V⁡(a,x) has no positive real zeros, and if a=32−2⁢n, n∈ℤ, then V⁡(a,x) has a zero at x=0.
