§8.19(viii) Analytic Continuation

ⓘ
Keywords:
analytic continuation, generalized exponential integral
Notes:
See Olver (1994b).
Permalink:
http://dlmf.nist.gov/8.19.viii
See also:
Annotations for §8.19 and Ch.8

The general function Ep⁡(z) is attained by extending the path in (8.19.2) across the negative real axis. Unless p is a nonpositive integer, Ep⁡(z) has a branch point at z=0. For z≠0 each branch of Ep⁡(z) is an entire function of p.
