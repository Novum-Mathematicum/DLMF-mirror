§27.3 Multiplicative Properties

ⓘ
Keywords:
completely multiplicative, completely multiplicative functions, multiplicative, multiplicative functions, multiplicative number theory, number-theoretic functions

Except for ν⁡(n), Λ⁡(n), pn, and π⁡(x), the functions in §27.2 are multiplicative, which means f⁡(1)=1 and

27.3.1 f⁡(m⁢n)=f⁡(m)⁢f⁡(n),
(m,n)=1.
ⓘ
Symbols:
(m,n): greatest common divisor (gcd), m: positive integer, n: positive integer and f⁡(n): multiplicative function

If f is multiplicative, then the values f⁡(n) for n>1 are determined by the values at the prime powers. Specifically, if n is factored as in (27.2.1), then

27.3.2 f⁡(n)=∏r=1ν⁡(n)f⁡(prar).
ⓘ
Symbols:
ν⁡(n): number of distinct primes dividing a number, n: positive integer, p,p1,…: prime numbers, ar: positive exponent and f⁡(n): multiplicative function

A function f is completely multiplicative if f⁡(1)=1 and

27.3.9 f⁡(m⁢n)=f⁡(m)⁢f⁡(n),
m,n=1,2,….

27.3.10 f⁡(n)=∏r=1ν⁡(n)(f⁡(pr))ar.
